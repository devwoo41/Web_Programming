#### HTML 문서

- HTML 문서는 태그로 표현되는 요소의 집합으로 구성되어 있다.

- 태그로 표현된 하나의 객체를 요소라고 한다.

- 태그에는 시작태그와 종료 태그가 있다.
ex. <html> ... </html> 로 표현되는 경우, 이는 html 태그의 시작과 끝을 나타내고 이 태그안에도 여러 태그(head, body 등등...)의 태그가 존재하고, 이러한 태그가 중첩된 형태로 존재한다.

#### 태그의 구성

- 태그는 태그 이름과 속성으로 구성되어 있다.
```
<img src="heart.jpg> width="100" height="50" alt="사랑합니다">
```
- 시작태그 + 내용 + 종료태그로 구성되어 있다. 

- HTML 태그는 대소문자 구분 없음

- 속성간에 불필요한 공백은 오류 발생

- HTML 내용은 보통 섹션이 나누어진다. 헤드 섹션과 바디 섹션으로 일반적으로 나누는데, 헤드 태그는 일반적으로 브라우저의 탭 부분에 해당하고, 바디 부분에는 실제 들어가는 내용에 해당한다.

#### 엔터티 코드

- 입력이 어려운 기호는 특별히 엔터티 코드값으로 표기한다.

#### 블록태그와 인라인태그

- 블록태그는 항상 새 라인에서 시작하여 출력하고, 양 옆에 다른 콘텐츠를 배치하지 않고 한 라인을 독점해서 사용한다. 
ex. <p></p>, <div>, <h1>등등...

- 인라인태그는 블록 속에 삽입되어 블록의 일부로 출력된다.
ex. <a>, <img>, <span>

#### img 태그

- src : 이미지 파일의 URL, ex. BMP, JPG, PNG... 등등
- alt : 이미지 출력에 실패할 경우 대체할 문자열
- width : 이미지의 가로 크기
- height : 이미지의 세로 크기

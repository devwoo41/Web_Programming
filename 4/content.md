#### 드롭다운 리스트

- 선택형 입력 콤보 박스 <select>를 통해 드롭다운 리스트를 만들 수 있다. 드롭다운 리스트는 한번에 모든 요소를 보여주는 것이 아닌 스크롤을 해서 내려서 목록을 보여주는 방식이다.
```
name : 요소 이름, 서버로 전송되는 폼 데이터의 이름
size : 콤보 박스 창 크기, 디폴트는 1
multiple : 다수 항목 선택 가능
```
- 항목을 나타내는 <option>
```
value : 요소 값, 버튼이 선택도니 경우 서버로 전송되는 폼 데이터의 값
selected : 이 속성이 있으면, 초기에 선택 상태로 출력
```

#### 캡션

- 캡션은 캡션 텍스트를 클릭해도 폼 요소를 클릭하는 효과를 내는 것이다.
